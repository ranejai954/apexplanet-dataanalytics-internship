This repository contains my work from the Data Analytics Internship at ApexPlanet Software Pvt. Ltd. It includes the complete implementation of data immersion and wrangling tasks, covering data cleaning, preprocessing, and feature engineering using Python and Pandas.

The project demonstrates handling real-world datasets by identifying missing values, correcting inconsistencies, and transforming raw data into an analysis-ready format. It also includes a structured data dictionary and well-documented code, making it a comprehensive reference for foundational data analytics workflows.

This repository serves as both a practical learning resource and a showcase of my ability to work with data, apply analytical thinking, and prepare datasets for further business intelligence and analysis.
