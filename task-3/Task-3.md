# Task 3: Video Game Sales Analytics Dashboard

This project focuses on analyzing a video game sales dataset to extract meaningful insights and present them through a structured dashboard. The objective was to move beyond basic analysis and apply key performance indicators (KPIs) to better understand market trends and performance.

The analysis was performed using Python (Pandas and Matplotlib), where important metrics such as total sales, average sales per game, top-performing genre, top platform, and average critic score were calculated. A deeper exploration was conducted on genre performance, platform distribution, and sales trends over time.

Based on the analysis, key insights were identified, such as the dominance of the Sports genre, the strong performance of the PS2 platform, and the presence of a long-tail distribution where most games have low sales. It was also observed that critic scores do not strongly influence sales, highlighting the role of external factors like popularity and marketing.

A dashboard was created to visually represent these insights, making the data easier to interpret and communicate. The project demonstrates the use of data analytics techniques to transform raw data into actionable insights and supports data-driven decision-making.
